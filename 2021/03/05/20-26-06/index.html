JavaScript笔记

Author: JiangHu295 Date: 三月 5, 2021  20:26:06 Category: JavaScript

JavaScript

1. 是一个基于浏览器的弱类型的基于对象的脚本语言

  1. 基于浏览器：JavaScript由浏览器来解析运行
  2. 弱类型：定义变量时，无需指定当前变量的类型，所以叫弱类型，而java 不然
  3. 基于对象：
    1. js中定义了诸多对象，供开发人员直接调用
    2. js中也有封装、继承的概念，但是没有多态
  4. 脚本语言：游戏和作弊脚本的关系

2.JavaScript代码往哪里写?

① 写在html文件中的script标记内（内联）

② 写在独立的js文件中（外链）

③ 可以写在HTML标记的事件中

3.JavaScript中的变量

语法: var 变量名 = 初始值;

案例 : var age = 10;

变量定义实际上是在内存中开辟一个空间,将值放在这个空间中进行管理,我们姑且将变量名当成访问这个空间的地址,即通过变量名访问其中的值

注意 : 变量有一个重要特点,就是变量的值可以改变

6.js中的函数

函数 = 方法, 在js中,方法(函数)是不能自己运行, 必须通过方法调用来执行

函数定义的方式如下:

  1. 通过function关键字定义 调用与声明函数的位置无关
  2. 函数表达式方式定义函数 先声明在调用
  3. 通过Function构造方法定义函数

7.js中的提升(hoist)

js中有一个比较特殊的功能,即函数和变量的提升

变量的提升 : 在定义的变量之前(之上)调用变量是被允许的,js会把这个变量的定义提升到调用之前,但是要注意,变量的提升不会给变量赋值的,仅仅是一个定义上的提升而已.

函数的提升 : 函数的提升允许开发人员在函数定义之前完成函数的调用

一定要注意 : 函数表达式定义的函数不会被提升.

8.js中的作用域(scope)

① local :

局部作用域,当前函数体,语句块作用域 , 语法通过var关键字定义 案例 : var a = 10;

② global :

全局作用域 , 全局作用域的属性会自动放在window对象下 , 访问时可以省略window对象

​ 语法 : 变量的定义不能用var定义

思考 : 在js中怎样可以访问到一个方法体中定义的局部变量呢？

③ 闭包(了解)

内部函数可以访问外部函数的闭包,可以有效的变量的全局污染.

9. JavaScript中的控制语句

1. 分支结构

  1. if结构

  2. if-else 结构

  3. switch-case结构

2. 循环结构

  1. while循环

  2. do-while循环

  3. for循环

    1. for(var 变量 in 数组或对象) 增强型循环

3.流程控制语句

  1. break语句 : 跳过本层循环，继续执行循环后面的语句
  2. continue语句 : 跳过本次循环中循环体剩余的语句而继续执行下一次循环
  3. return语句 : return 语句只能用于函数中，作用效果比 break 语句更加强悍，将会直接终止当前函数
                        执行，包括循环后面的语句也不再执行

10. JavaScript中的DOM对象

1. 什么是DOM对象

  1. 动态操作HTML元素 : 利用dom对象对html元素的新增和删除
  2. 可以对html元素的属性进行操作, 包括添加属性,修改属性值,删除属性
  3. 可以对html元素的css样式进行操作. 比如设置一样样式等
  4. 可以对html元素的事件进行响应.

2. 创建dom对象

  1. 通过html元素的id属性值,获取dom对象–getElementById
  2. 根据html元素的name属性值获取dom集合数组,需要注意的是,getElementsByName 方法返回的是dom对象数组,而getElementById 方法返回的是单一的一个dom对象
  3. 通过html元素的元素名称获取对应的dom对象,注意 : 返回的是一个dom对象数组—getElementsByTagName
  4. 通过选择器的方法获取dom对象—①querySelector ②querySelectorAll

3. 获取 | 设置html元素的值

1. 通过dom对象的innerHTML属性
 2. 通过dom对象的innerText属性

4. dom动态操作元素

利用dom对象可以创建一个新的html元素,并且可以把新的html元素添加到指定的标记内.

5.html元素的常用事件

html元素的事件,是html元素与js进行交互操作的一种途径.

1. 鼠标相关事件
  1. onclick : 单击事件
  2. onmouseenter | onmouseover : 鼠标进入事件
  3. onmouseleave | onmouseout : 鼠标离开事件
  4. ondbclick : 双击事件
  5. onmousedown : 鼠标按下事件
  6. onmouseup : 鼠标弹开事件
2. 成功载入事件

onload : 装载完毕事件,即浏览器在成功的将所有html元素装载(运行)完毕后触发.

​ 所以在这个事件中通过 document.querySelector(‘.div1’) 创建dom对象不会返回null

3. 表单相关事件
  1. onblur : 失去焦点事件, 在表单控件失去焦点时触发
  2. onsubmit : 表单提交事件 ,即点击了提交按钮. 在当前表单提交时触发return boolean 的使用

11. JavaScript中的常见对象

1. 正则表达式对象

​ 是 js 中提供的功能及其强大的模式匹配工具

​ 语法 : /^…$/

​ 案例 : /^[a-z]$/; /*代表一个字符，并且这个字符是小写字母/^[a-z] */

[]：代表1个字符 /^[a-z]$/；

[abc]：必须是一个字符，并且这个字符要么是a 要么是 b 要么是c之一

[a-z]：小写字母

[A-Z]：大写字母

[0-9]：一个数字

[0-9a-zA-Z] ：必须是一个字符，要求是数字或大小写字母

{}：代表字符的个数

{5}：5个字符

[a-z]{5} : 5个小写字母

[0-9]{6}：6个数字

[0-9]{6，8}：6-8位数字

[0-9]{6，}：至少6个数字

元字符：

\d:数字 == [0-9]

\n : 换行
