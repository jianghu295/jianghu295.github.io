0113_盒子模型&定位&浮动

Author: JiangHu295 Date: 三月 5, 2021  20:31:24 Category: JavaScript

盒子模型

  1. 边框属于盒子边缘，边框里边属于盒子内部，边框的大小会影响整个盒子的大小
  2. 一共个盒子的可见框的大小，由 内容区 内边距 和 边框共同决定
  3. 内边距的设置也会影响盒子的大小
  4. 外边距不会影响盒子的大小，但外边距会影响盒子的位置，margin 会影响盒子实际占用空间
  5. 相邻的垂直元素外边距会发生重叠现象
  6. 父子元素减相邻外边距，子元素的会传递给父元素(上外边距)，父子外边距的折叠会影响到页面的布局，必须要进行处理
  7. 行内元素不支持设置宽度和高度
  8. 行内元素可以设置padding，但垂直方向padding不会影响布局
  9. 行内元素可以设置border，垂直不影响
  10. 行内可以设置margin，垂直不影响

其他

  • display 用来是指元素显示类型

    • 可选值
      • inline 行内元素
      • block 块元素
      • inline-block 行内块元素 可以设置宽度和高度，又不会独占一行
      • table 表格
      • none 不在页面中显示

  • visibility 设置元素显示类型

    • 可选值
      • visible 默认值，正常显示
      • hidden 在页面中隐藏不显示，但是依然占据页面位置

  • 盒子尺寸

    • box-sizing
      • 可选值
        • content-box 默认值，宽度用来设置内容区的大小
        • border-box 宽度和高度用来设置整个盒子可见框的大小(width 和 height 指的是内容区和内边距和边框的总大小)
