1. String类

1.1 String类的构造方法

  1. 创建一个空字符串对象

    String str = new String();

  2. 以指定的字符创为副本，创建一个字符串对象

    String str = new String("abc");

在该例子中存在两个字符串对象，一个对象是==字符创常量==，一个对象是==字符串变量str==

  1. 将字符数组转换成字符串对象的方法

    char c[]={‘a’,’b’,’c’};String s=new String(c);

  2. 将byte类型的数组转换成字符串

    byte b[]={65,66,67};String s=new String(b);

注意

​ 利用数组创建String对象时，复制的是数组的内容，数组的改变并不影响String对象

1.1 String

常量放在前面，变量放在后面，避免空异常

  1. String类位于（lang）包下 不用导包

  2. 计算字符串长度的方法是（**length()**）

  3. 将简单类型的变量与字符串连接时，将会自动调用（String）类的（valueOf）方法，把简单类型数据转换为字符串之后再做连接

  4. 将对象与字符串连接时，将会调用（String）类的（tostring）方法，把对象转换为字符串之后再做连接， 输出格式为：类名@6位十六进制数

    该方法在Object类中存在，String类中对该方法进行了重写，若用String对象调用toString（）方法，调用的是String类中的该方法，若用其他对象调用toString（）方法，调用的是Object类中的该方法

  5. 根据功能要求填写所要使用的方法

    a) 返回指定位置上的字符（**charAt(int)**）

    b) 截取指定范围的子串存放到字符数组中（**getChars(start,end,char[],start)**）

    c) 截取指定范围的子串（**subString(int) / subString(start,end)**）

    d) 将字符串转换为字节数组（**getBytes()**）

    e) 将字符串转换为字符数组（**toCharArray()**）

    f) 以区分大小写的方法比较两个字符串的内容是否相等（**equals()**）

    **==**：比较两个字符串对象是否为同一个对象，即比较两个对象所存放的地址

    注意：equals（）的特殊用法

    例如一个登录窗口，存在一个输入用户名文本框，若声明一个String类型的name来存放该文本框中的用户名内容时，当用户名没有填写，而用代码name.equals(“Marina”)判断用户填写的用户名是否等于指定的“Marina“时，将会出现空指针异常，这种情况下，应该将判断的代码写成“Marina”.equals(name)

    g) 以不区分大小写的方法比较两个字符串的内容是否相等（**equalsIgnoreCase()**）

    h) 比较两个字符串的大小关系（**compareTo()**）

    i) 检查一个字符串是否以另一个字符串开头（**startsWith()**）

    j) 检查一个字符串是否以另一个字符串结尾（**endsWith()**）

    k) 在一个字符串中搜索某个字符第一次出现的位置（**indexOf()**）,没有找到，则返回-1

    l) 在一个字符串中搜索某个字符最后一次出现的位置（lastIndexOf）

    m) 将一个字符串的末尾连接上指定的字符串（**concat()**）

    n) 将一个字符串中指定的子串换成另一个子串（**replace(s1,s2)**）

    o) 去掉字符串两端的空格（**trim()**）

    p) 将字符串全部转换为大写（toUppercase）

    q) 将字符串全部转换为小写（toLowercase）

    r）split（字符串）：将一个字符串对象以参数指定的字符进行分解，将分解出来的每个子串存入一个字符串数组中

    方法描述
    intern如果常量池中存在当前字符串, 就会直接返回当前字符串.
    如果常量池中没有此字符串, 会将此字符串放入常量池中后, 再返回

2.1 数值包装类

原始数据类型包装类
charCharacter
byteByte
shortShort
intInteger
floatFloat
doubleDouble
longLong
booleanBoolean

1． 包装类的作用是将（基本）数据包装成（引用数据类型/类类型）

2． 8种包装类分别是（Byte）（Character）（Integer）（Long）（Float）（Double）（Boolean）（Short）

3． 6种数值型的包装类都有一个统一的抽象父类是（Number），该类中定义了6个返回不同类型数值的抽象方法，分别是（**byteValue()）（intValue()）（shortValue()）（longValue()）（doubleValue()）（floatValue()**），这些方法的功能是将包装类的对象转换为（）的数值

4． 包装类中有一个静态方法，用于将字符串转换为基本类型的数据，这些静态方法分别是（**parseInt()）（parseByte()）（parseShort()）（parseLong()）（parseFloat()）（parseDouble()）（parseBoolean()**）

5． 包装类中还存在一个静态方法，用于将字符串转换为一个基本数据类型，这个静态方法是（**parseInt…**）

6 . 在包装类中，存在方法**valueOf()**，可以将一个字符串或原始类型的数据转换为一个包装类的对象

​ 使用格式：包装类 对象=包装类.valueOf(参数)
