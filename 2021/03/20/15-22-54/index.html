输入输出流

（一）File对象

1. 方法

  • 构造方法

    • File(String filename)

  • 普通方法

    • 方法描述
      boolean exist()判断文件是否存在
      String getName()获取文件名
      String getPath()获取文件路径
      boolean isDirectory()判断是否为目录
      boolean isFile()判断是否为文件
      boolean isHidden()判断是否为隐藏文件
      long lastModified()文件最后一次修改时间
      String[] list()返回文件对象代表的路径下的文件和文件夹
      File[] listFiles()返回文件对象代表的路径下的文件和文件夹
      long length()文件的长度

匿名类

练习 递归

（二）流的分类

流向分

  • 输入流：文件–>内存 读文件
  • 输出流：内存–>文件 写文件

传输的内容上划分，流分为

  • 字节流：8位 不能处理文字 读文件的时候中文乱码
  • 字符流：16位 Unicode 在字节流上包装得到 可以处理字节流 也可以处理汉字 读文件的时候中文不乱吗
