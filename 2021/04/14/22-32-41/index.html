Git 创建仓库

git init

Git 使用 git init 命令来初始化一个Git仓库

指定目录作为Git仓库
git init newrepo

git clone

使用 git clone 从现有 Git 仓库中拷贝项目
git clone <repo>

克隆到指定的目录
git clone <repo> <directory>

Git 的设置使用 git config 命令，显示当前 git 配置信息

编辑 git 配置文件
