1. 类集框架

  • Java的集合框架使你的程序处理对象组的方法标准化
  • 集合框架被设计的目的：
    • 这种框架是高性能的。对基本集合类（动态数组，链接表，树和散列表）的实现是高效率的
    • 框架必须允许不同类型的集合以相同的方式和高度互操作方式操作
    • 集合是容易扩展和或修改的
  • Java平台提供了一个全新的集合框架。“集合框架”主要由一组用来操作对象的接口组成,位于java.util包下。不同接口描述一组不同数据类型
  • Collection接口——定义了存取一组对象的方法，将对象以一个接一个的方式存储，其子接口Set和List分别定义了具体的存储方式
    • Set中的数据对象没有顺序且不可以重复
    • List中的数据对象有顺序且可以重复

1. String类

1.1 String类的构造方法

  1. 创建一个空字符串对象

    String str = new String();

  2. 以指定的字符创为副本，创建一个字符串对象

    String str = new String("abc");

在该例子中存在两个字符串对象，一个对象是==字符创常量==，一个对象是==字符串变量str==

  1. 将字符数组转换成字符串对象的方法

    char c[]={‘a’,’b’,’c’};String s=new String(c);

  2. 将byte类型的数组转换成字符串

    byte b[]={65,66,67};String s=new String(b);

注意

​ 利用数组创建String对象时，复制的是数组的内容，数组的改变并不影响String对象
